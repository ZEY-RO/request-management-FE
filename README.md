This repository contains the React frontend for the Request Management System.

Prerequisites
- Node.js 16 or newer (Node 18+ recommended)
- npm (or yarn/pnpm)
- Git

Quick start
1. Clone the repo

```bash
git clone https://github.com/ZEY-RO/request-management-FE
cd request-management-FE
```

2. Install dependencies

```bash
npm install
# or
yarn
# or
pnpm install
```

3. Run the development server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open http://localhost:5173 in your browser.

Build for production

```bash
npm run build
# or
yarn build
# or
pnpm build
```

Preview the production build locally

```bash
npm run preview
# or
yarn preview
# or
pnpm preview
```
